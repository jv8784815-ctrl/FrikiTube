Introducción

FrikiTube es una API HTTP construida con Express.js que actúa como intermediario para descargar contenido de YouTube. Recibe una URL y calidad, y devuelve un link directo al archivo procesado.

Todos los endpoints aceptan POST con body en application/json. Las respuestas siempre son JSON.

Base URL https://tu-api.onrender.com
02
POST /api/ytmp3

Procesa un video de YouTube y devuelve un link de descarga directo al archivo MP3 en la calidad solicitada. Ideal para el comando .play o .yta de tu bot.

Parámetros del body
Campo Tipo Estado Descripción
url string requerido URL del video de YouTube
quality string opcional Default: 320k
128k · 192k · 256k · 320k
Request
Node.js — Baileys
const axios = require('axios'); async function ytAudio(url) { const { data } = await axios.post( 'https://tu-api.onrender.com/api/ytmp3', { url: url, quality: '320k' } ); return data; // objeto completo }
cURL
curl -X POST \ https://tu-api.onrender.com/api/ytmp3 \ -H "Content-Type: application/json" \ -d '{ "url": "https://youtu.be/xxxx", "quality": "320k" }'
